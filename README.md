## Getting Started

create a `.env` file with the following variables:
```json
MONGO_URL=###
PINECONE_ENVIRONMENT=###
PINECONE_API_KEY=###
OPENAI_API_KEY=###
```

First, run the development server:
```bash
npm install
npm run dev
```
